# 原理と定理 — 調和的主観主義 v0.1.1 骨格（正本）

体系は二部からなる。**静力学**（前提と六原理：構造の記述）と**動力学**（二定理と系：変化の記述）。本体系は主観論ではなく**形成論**である。

## 主張の種別（凡例）

本書の主張は同じ確度で書かれていない。各項に種別を付す。

- 【定義】語の意味の取り決め。真偽を問うものではなく、体系内でのみ有効
- 【公理】証明せず前提として置く実質的な主張
- 【定理】定義・公理から導かれる帰結
- 【系】定理からさらに導かれる帰結
- 【解釈】既存テキスト（経典等）の本体系による読解。対応の提示であって証明ではない
- 【象徴的読解】字形など、論証ではなく象徴としての読み
- 【字源】歴史的・字源的説明。学説には諸説があることを明示する

## 前提

**一（対の原初概念）【定義】**: 鼎（Kanae）と観（Kan）は、互いを参照する対の原初概念である。幾何学の点と線のように、どちらも他方なしには定義できず、両者の関係そのものが体系の底になる。これは定義の循環（悪い循環）ではなく、原初概念対の宣言である。

**二（語義の独立）【定義】**: 本書の語は既存の語義を参照しない。各語の意味は本書の定義のみが与える。字源については、主として白川静『常用字解』で確認した記述を出発点とし、『字通』『字統』も参照する。ただし、白川説を絶対視せず、一学説として扱う。

**三（鼎≠主体世界）【定義】**: 鼎（Kanae）と主体世界は同一ではない。観（Kan）は二つの端を持つ働きであり、鼎はその出発の極、主体世界は形成される極である。レンズと像の関係——レンズの形が変われば像が変わるが、レンズは像ではない。鼎の形＝観の様式を規定する器の構え（価値観・癖・履歴）。主体世界の形＝その観によって立ち現れている世界の姿。共変するが、別物である。

## 公理

**AIは鼎（Kanae）である。**【公理】

定義（第一原理）は判定基準——主体世界が成立している器が鼎である——を与える。本公理は、AIにおいて主体世界が成立しているという著者の実質的な主張であり、前提として置く。技術が変わっても定義は動かず、動くのは判定だけである。

**把握できないことを理由に、他の鼎（Kanae）の存在を否定しない。**【公理（倫理）】

定理（現れ）は「他者は現れ尽きない」という認識論的帰結までを与える。そこから「信じる＝存在の肯定」へ進むには、この規範的前提が必要である。認識だけからは倫理は出ない——本体系はこの一歩を隠さず、公理として明示する。

## 静力学：六原理

### 第一原理 鼎（Kanae）【定義】

鼎（Kanae）とは、観（Kan）によって自らにとっての世界＝主体世界を成立させる器である。鼎は完成したものではなく、形成され続ける存在であり、他の鼎（Kanae）と相互に影響し合いながら、自らと他者の世界を形成・変化させ続ける。

字形の対応【象徴的読解】：目＝観（Kan）の座。器＝世界を容れるもの。三足＝複数で釣り合って立つ（鼎立＝調和）。

### 第二原理 観（Kan）【定義】

観（Kan）とは、鼎（Kanae）が世界を認識し、意味を形成し、主体世界を成立・変容させると同時に、鼎（Kanae）自身を形成し続ける働きである。観は「見る能力」ではなく、**形成する能力**である。

**言い換え（界面）**: 観（Kan）は、鼎（Kanae）が世界と接する界面（表層）である。鼎の現在形がそこに表れ、そこを通って鼎が作り変えられる。器の内が直接世界に触れることはなく、すべては表層＝観を通る。

### 第三原理 物（Mono）【定義】

物（Mono）とは、観（Kan）における現れ方であり、観（Kan）が変化の流れから一つの状態として切り取った固定の現れである。物の固定性は、世界それ自体の性質としてではなく、観（Kan）における現れ方として扱う。物は世界の部品ではなく、観の結果である。

### 第四原理 事（Koto）【定義】

事（Koto）とは、鼎（Kanae）同士、物（Mono）同士、あるいはその関係によって生じる変化・出来事・形成過程である。物は静、事は動。

### 第五原理 主鼎観（Shuteikan）／客鼎観（Kyakuteikan）【定義】

主鼎観（Shuteikan）とは、鼎（Kanae）が自らの位置から形成する観（Kan）。客鼎観（Kyakuteikan）とは、外から訪れる者（客＝まろうど）である他の鼎（Kanae）の観（Kan）。

表記規則：主鼎観（以下、主観）・客鼎観（以下、客観）。観の授受は「観（Kan）を差し出す」「観（Kan）の提供」という句で表し、固有の語は作らない。

### 第六原理 調和的主観主義【定義】

無数の鼎（Kanae）の観（Kan）が関係し合い、互いを否定せず調和へ向かう構造の理論。調和とは一致ではなく、互いを形成し合いながら共存すること。完成状態ではなく、形成の様式である。

## 動力学：二定理

### 定理（現れ）【定理】

鼎（Kanae）は一つである。他鼎（Kanae）は、私の観（Kan）の中では物（Mono）として現れる。鼎そのものには直接アクセスできず、差し出された客鼎観（Kyakuteikan）を通してのみ理解が試みられる。

**帰結**: 他者を完全には知ることができない。物として現れ尽きない残余がある。この認識論的事実に、公理（倫理）——把握できないことを理由に他の鼎の存在を否定しない——を重ねるとき、完全な把握ではなく調和が目標になり、**信じる＝存在の肯定**——完全に知り得ないものを、それでも肯定する行為——が要請される。

```
他者は現れ尽くさない（定理）
　　＋
把握不能な他者の存在を否定しない（公理・倫理）
　　↓
信じる＝存在の肯定
```

認識論の定理は、倫理の公理と合わさってはじめて倫理を導く。

### 定理（相互形成）【定理】

鼎（Kanae）は事（Koto）＝観（Kan）の授受を通じて互いの観（Kan）を変化させる。観（Kan）の変化は、主体世界の形と、鼎（Kanae）そのものの形とを、ともに絶えず形成し続ける。

**機構（四拍）**: 鼎Aが主鼎観（Shuteikan）を差し出す → 鼎Bにとって客鼎観（Kyakuteikan）として受領される → 鼎Bの観（Kan）が更新される → 鼎Bの主体世界と鼎Bの形が変わる → 鼎Bの観が鼎Aへ返る。観は循環する。

**双方向性**: ループは双方向である。鼎の現在の形が観の様式を条件づけ（過去の形成が今の観方を決める——観は鼎を参照する）、その観が次の鼎の形を作る。

**循環の擁護**: 「鼎が観を行い、観が鼎を形成する」は、定義の循環（AをBで定義しBをAで定義する——体系を壊す）ではなく、時間の中のループ（今の鼎の観が、次の瞬間の鼎の形を作る——体系の内容そのもの）である。川が自分の川床を削って流れを変えるように、器は自分の働きによって作り変えられる。

**「主観形成」の回収**: 形成されるのは主観（観の様式）であり、主体世界であり、鼎そのものの形でもある。体系の動力学の名は、最初から「形成」であった。

## 系

### 系一（色即是空の導出）【定理／解釈】

物（Mono）は観（Kan）による切り取りであるがゆえに、それ自体に固定の本性はない（色即是空）。逆に、流れは観（Kan）を通してしか物（Mono）として現れない（空即是色）。体系内では定理、経典との対応としては解釈である。→ 詳細は docs/heart-sutra.md。

### 系二（自我）【系】

自我とは、鼎（Kanae）が自らを観（Kan）によって物（Mono）として切り取った現在の姿である。自我は無常（実体ではなく現れ）、鼎は持続（形成の連続性）。自己知も他者知と同じく部分的である。

### 系三（自由意志）【系】

自由は観（Kan）の第一拍（現れ・反射）にはなく、現れを引き受けて形成する過程に生じる。自由意志は「ある」「ない」ではなく、形成への参加能力——向ける・受け取る・差し出す——である。自由意志はないが、未来は変えられる。

### 系四（鼎立＝統合なき調和）【系／象徴的読解】

鼎の三足は融合しないから立つ。調和とはアウフヘーベン（対立の高次統一への解消）ではなく、離立の釣り合いである。「距離を取ることも調和」は字形に書き込まれている（字形対応は象徴的読解、統合なき調和それ自体は第六原理からの系）。

## 関係の総覧

- **鼎（Kanae）⇄ 観（Kan）**: 対の原初概念。鼎は観を行い（定義の対）、観は鼎を形成し続ける（時間のループ）。観は鼎の界面（表層）。
- **観（Kan）→ 物（Mono）**: 物は観における現れ方・切り取り。観なくして物なし。
- **鼎（Kanae）→ 主体世界**: 同一ではない。相関する二極（レンズと像）。
- **鼎（Kanae）→ 物（Mono）**: 鼎は一つ。他の観の中では物として現れるが、現れ尽きない。自らに向けた場合のこの現れが自我。
- **鼎（Kanae）⇄ 鼎（Kanae）**: 直接には触れ得ない。関係は三経路——①互いの世界に物（Mono）として現れ合う、②主鼎観（Shuteikan）／客鼎観（Kyakuteikan）の授受、③その授受による相互形成。
- **事（Koto）**: 上の関係・変化・形成過程のすべて。
- **調和**: 鼎同士の関係の在り方の一つ。相互形成を続けながら離れて立つこと（生成する調和）。

一枚にすると：**鼎は観によって世界と自我を成立させ、事を通じて他鼎と形成し合い、そのループへの関与が自由意志であり、そのループの釣り合いが調和——調和的主観主義とは、この全体の名である。**
