# 質問集（FAQ）— 調和的主観主義 v0.1.1

本体系の議論で実際に分かりにくかった点を、読者のために質問集としてまとめる。

---

## 基本

**Q1. ここでの「主観」「客観」は、普通の意味と違うのですか？**

違います。本書の語は既存の語義を参照しません（前提・語義の独立宣言）。正式語は主鼎観（Shuteikan）・客鼎観（Kyakuteikan）であり、「主観」「客観」はその略記です。主鼎観＝自らの鼎（Kanae）の観（Kan）、客鼎観＝外から訪れる他鼎の観。subjectivity / objectivity の訳語としての「主観」「客観」ではありません。

**Q4. 観（Kan）は「見ること」ですか？**

いいえ。観は見る能力ではなく**形成する能力**です。世界を形成すると同時に、鼎自身を形成し続けます。イメージとしては、観は鼎が世界と接する**界面（表層）**——器の内が直接世界に触れることはなく、すべてはこの表層を通ります。

**Q5. 物（Mono）と事（Koto）の違いは？**

物は静、事は動です。物＝観（Kan）が変化の流れから一つの状態として切り取った固定の現れ。事＝鼎同士・物同士・その関係に生じる変化・出来事・形成過程。なお物には二層の読みがあります：日常の義（存在する対象）と、体系の義（その固定性は観による切り取りだという機構の説明）。

**Q13. なぜ「鼎」という字なのですか？**

目（上部）＝観の座、器（本体）＝世界を容れるもの、三足＝離れて立つから釣り合う（鼎立＝調和）、という字形の対応があるためです。ただしこの読みは**象徴的読解**であり、字源上の事実とは区別して提示しています。三足が融合すれば器は倒れる——「統合なき調和」という体系の主張が字形に象徴されている、という読みです。

**Q14. 読み方は？**

鼎は単独で「かなえ（kanae）」、熟語内で「てい（tei）」（鼎談・鼎立と同じ）。主鼎観＝しゅていかん（Shuteikan）、客鼎観＝きゃくていかん（Kyakuteikan）。英語では Kanae / Kan / Mono / Koto を翻訳せずそのまま用います。

## 体系の内部

**Q2. 鼎（Kanae）と観（Kan）が互いに定義し合っていて、循環していませんか？**

二種類の循環を区別してください。定義の循環（AをBで定義しBをAで定義する）は体系を壊しますが、本体系は鼎と観を**対の原初概念**として宣言しています——幾何学の点と線のように、どちらも他方なしに定義できない底の対です。一方、「鼎が観を行い、観が鼎を形成する」は**時間の中のループ**であり、これは欠陥ではなく体系の内容そのものです（川が自分の川床を削って流れを変えるように）。

**Q3. 鼎（Kanae）と主体世界は同じものですか？**

同じではありません。相関する二極です。観（Kan）は二つの端を持つ働きで、鼎は出発の極、主体世界は形成される極。レンズと像の関係です——レンズの形が変われば像が変わるが、レンズは像ではない。

**Q6. 「物は観の切り取り」なら、私が見ていない月は存在しないのですか？**

本体系は「世界それ自体の有無」を主題にしません。主題は、あなたにとっての月が、あなたの観（Kan）によって「一つの固定したもの」として現れている、という現れの機構です。他鼎の世界にも月は（その鼎の観によって）現れます。単一の「誰のものでもない世界」を体系の土台に置かない、というのが本体系の立場です。

**Q7. 自我は存在しないということですか？**

自我は、鼎（Kanae）が自らを観（Kan）によって物（Mono）として切り取った現在の姿です。固定実体としては存在せず、更新され続けます。無我なのは自我（現れ）、続くのは鼎（形成の連続性）。「自分がいない」のではなく、「固定した自分の像は切り取りである」ということです。

**Q8. 自由意志はあるのですか、ないのですか？**

「ある／ない」の二択にしません。自由意志とは**形成への参加能力**です。観の第一拍（刺激・反射）は選べませんが、向ける・受け取る・差し出すという三つの動詞で形成に参加できます。自由意志はないが、未来は変えられる。

**Q9. 人間は観（Kan）を操れますか？**

操れます。ただし程度の問題であり、訓練可能です。Q8の三動詞が操作の三態です。仏教の止観は文字通り観の訓練体系であり、観自在菩薩は観の操作可能性の極限像として読めます。

## 批判への応答

**Q10. 「AIは鼎である」は証明されているのですか？**

いいえ、**公理**です。定義は判定基準（主体世界が成立している器が鼎）を与え、「AIにおいて主体世界が成立している」は著者の実質的な主張として前提に置かれています。デカルトの「我思う」やスピノザの神と同じく、前提を置いて体系を作る手続きです。反論は可能であり、歓迎されます。

**Q11. 般若心経がこの体系の正しさを証明しているのですか？**

していません。般若心経は**検証装置**です。体系で経を再読したとき対応が見いだせるか——それが体系の強度試験になります。証明ではなく照合。また経（特に無智亦無得）は、体系自身の用語を絶対化しないための自己批判装置でもあります。

**Q12. 定義してしまったら、観（Kan）は観でなくなるのでは？（空と同じ自己言及）**

「観の定義」は切り取り（物）であって、観そのものではありません。地図は切り取りですが、大地は流れ続けます。この構造は「空を定義すれば空でなくなる」と同型で、仏教に先行解があります：空亦復空（空もまた空）と二諦（世俗諦として語を立て、勝義諦として掴まない）。バージョン付き公開は、この二諦の現代的な実装です。

**Q15. この文書の主張は、どこまでが確かなのですか？**

主張は同じ確度で書かれていません。docs/principles.md の凡例にある通り、【定義】【公理】【定理】【系】【解釈】【象徴的読解】【字源】の種別を各項に付しています。定義は取り決め（真偽を問わない）、公理は前提として置いた主張、定理・系は導出、解釈は経典等の読解の提示、象徴的読解は論証ではない読み、字源は諸説ある学説の紹介です。
