## 本體系的性格

**本體系並非「主觀論」，而是「形成論」。**

其中心命題並不是「人形成世界」。而是**「透過形成世界，鼎（Kanae，中文讀音 dǐng）本身也被形成」**，更進一步是**「鼎（Kanae）彼此相互形成」**。世界並非已然完成之物，而是持續被形成。因此，調和並非一種完成的狀態，而是**形成的樣式**。

## 語義的獨立宣言（最重要的前提）

**本書所用的語詞，不參照辭典式、哲學史式的既有語義。各語詞的意義，僅由本書的定義給予。**

關於字源，主要以白川靜《常用字解》所確認的記述為出發點，並參照《字通》《字統》。惟不將白川之說視為絕對，而作為一家學說看待。

## 表記規則（全文書共通）

- 鼎（Kanae）・觀（Kan，中文讀音 guān）・物（Mono，中文讀音 wù）・事（Koto，中文讀音 shì），於定義、初次出現時必定並列漢字與羅馬字。
- 鼎的讀音：單獨時為 **kanae**，於複合詞內為 **tei**（與鼎談、鼎立同樣的連音變化）。
- 正式語與略記：於開頭宣告 **主鼎觀（Shuteikan）／以下簡稱主觀**・**客鼎觀（Kyakuteikan）／以下簡稱客觀**，此後的「主觀」「客觀」皆作為略記解讀（此處的主觀、客觀並非 subjective／objective 之譯語）。
- 不為「觀的提供」「觀的受領」另造專用語。以「遞出觀（Kan）」「觀（Kan）的提供」這樣的語句表示（採取不增加難解語詞的方針）。
- 英語版：Kanae / Kan / Mono / Koto / Shuteikan / Kyakuteikan 不予翻譯，於初次出現時附上定義。英語略記為 Self-Kan / Other-Kan。體系名統一為 **Harmonious Subjectivism**。
