# 思想史定位 — 調和的主觀主義（Harmonious Subjectivism） v0.1.1

方針：不將既有的用法斷定為「錯誤」，而是宣告「本書如此定義」。關於字源，主要將白川靜《常用字解》中經確認的記述，定位為**作為出發點而採用的一種學說**，並參照《字通》《字統》。不將白川之說絕對化，並列通說（形聲・音符說）的存在，以避免不必要的字源爭論。

---

## 一、翻譯語的歷史——為何需要重新定義

「主觀」「客觀」作為當今的哲學用語，是十九世紀後半在日本作為 subject / object 的譯語而確立，並傳播至中國（主观／客观）・韓國（주관／객관）。在日中韓三地，皆固定為「主觀＝subjective（個人的看法）」「客觀＝objective（無論誰看都相同）」。

然而字義另當別論。主觀＝由主來觀。客觀＝由客（自外而來的他者）來觀。本體系回歸字義。漂移的是熟語，而非字（客人・客體仍保有字義）。

subjective / objective 的語源也經歷了同型的變化：subject ＝ sub-icere（向下投擲→基體→認識主體），object ＝ ob-icere（向前投擲→認識對象）。在西方，意義同樣從「主體／對象」轉移至「主觀的／客觀的」。因此本體系的主鼎觀（Shuteikan）・客鼎觀（Kyakuteikan），並非 subjective / objective 的重新定義，而是作為另一種新概念而確立。

## 二、與西田幾多郎的差異（最重要）

| | 西田 | 本體系 |
|---|---|---|
| 出發點 | 主客未分（純粹經驗）→ 由此主客分離 | 鼎（Kanae，中文讀音 dǐng）作為前提而存在，透過觀（Kan，中文讀音 guān）使主觀被形成・更新 |
| 善的根據 | 置於純粹經驗（主客未分） | 並非源於反射，而是產生於接收反射並加以處理的過程 |
| 終極的場所 | 絕對無的場所（單一的根據） | 不立單一的根據。有多少鼎就有多少主體世界（多元的徹底化） |
| 主體的生成 | 論述主客誕生之前 | 鼎並非被生成，而是在持續存在的同時改變其形（形成的連續性） |

不造新語、而以重新定義既有語彙將日本哲學推向世界的西田路徑（pure experience / basho），也是 Kanae / Kan 流通策略的先例。

## 三、與布伯・列維納斯的關係

「我與汝」的關係結構（唯有在關係之中，我才成為我）是獨立達致的。不過本體系固有的要素——親鸞・諸法無我的整合、將神當作功能來處理的立場、有多少主觀就有多少世界這種明示的多元性、終端的犧牲——布伯並不具備。並非「布伯的再發現」，而是「包含布伯式結構、更為寬廣的體系」。

## 四、與黑格爾的對立——鼎立的字形化

黑格爾的辯證法必然將對立揚棄（Aufheben，止揚）而趨向更高層次的統一。本體系則以「保持距離亦是調和」明示地拒絕統合。

此反黑格爾式的結構，早已作為物理被鼎（Kanae）的字形所預示：三足因不融合而得以站立。若融合為一點，器便傾倒。調和＝離立的均衡，而非統合。

## 五、哈洛威（Haraway，situated knowledge／處境知識）——客觀側的佐證

哈洛威否定了傳統的客觀性＝「從並非任何處之處看見一切的神之把戲（god trick）」，並提出「具有身體的客觀性」「從某處而來的眺望（view from somewhere）」。**將「客觀」這個語詞本身，重新定義為表明立場後所遞出的、部分性的觀**——現代認識論的一個主要流派，獨立地踏出了這同樣的一步。這是客鼎觀（Kyakuteikan）＝「從某處而來之視點的提供」的先行近接概念。

結論：既有的客觀（view from nowhere）與本體系的客觀並非「兩種對等的意義」，而是作為**位階不同的兩層**——實際存在的是具有位置之觀的提供，view from nowhere 則是理想或虛構——而提出。

## 六、egophoricity（自我性）——主觀側的佐證

在部分語言（藏語・尼瓦爾語等）中，「這是我以特權方式知曉之事」這種自我的涉入，會以動詞的形式被義務性地標記（egophoricity）。在陳述句中「認識的權威」歸於說話者，在疑問句中則移轉至聽話者（shiftability）——文法正在演示主觀⇄客觀的切換。這是「由主體所言則為主觀、由他者所言則為客觀」這種區分，至少在部分語言的文法中確實存在的證據。

## 七、與佛教・唯識的關係

- 能取（grāhaka，捕捉的一方）／所取（grāhya，被捕捉的一方）看似主觀／客觀，但與本體系的客觀（他者之觀）不同（所取是「對象」，而非「他者的認識」）。並不存在與本體系的客鼎觀嚴格對應的佛教語彙。
- 唯識將能取・所取的二元性本身視為虛構，這一點與本體系「主觀與客觀皆非固定實體、而是變化的」相近。
- 想（saṃjñā：識別特徵、賦予名稱並再度認識的作用），與物化（想的相位）幾乎對應。
- 梅洛龐蒂的「觀看的身體／被觀看的身體」，在鼎之一體性（鼎為一，物為顯現方式）的形式化之際曾被參照。

## 八、生成認知論（enactivism）

認識並非外界的摹寫，而是生物在與環境相互作用之中，使對該生物而言的世界得以顯現（enaction）——這是與「透過觀（Kan）使主體世界成立」最為接近的既有思想。不過生成認知論著重身體行為與環境的相互作用，本體系則進一步包含透過觀的授受所形成的、鼎與鼎之間的相互形成。

## 九、與白川靜的關係（引用與擴張的界線）

- **白川的說明（已於《常用字解》一次確認）**：「觀（觀）」為形聲，音符為雚。雚是鸛（白鸛），被視為神聖之鳥，用於鳥占。「以雚進行鳥占、察知神意，即『看、審視』之事，大概便稱之為觀」。原本大概是就農耕儀禮而察看神意，其後廣泛用於「看、細看」，也用於能眺望遠方的高聳建築（道觀＝道教的寺院）。
- **與通說的差異（並列記載）**：白川與通說在字形分類上皆為形聲・音符雚，此點一致。分歧在於對雚的解釋——白川賦予其神聖之鳥・鳥占這種咒術性意義的來歷，通說（《說文解字》「諦視也」等）則將雚視為單純的音符，不承認咒術性意義。學術上有批評指出，白川的咒術性解釋欠缺文獻與考古的根據。
- **本體系的擴張**：將白川所示的「察知意義的作用」，不侷限於宗教脈絡，擴張為主體認識世界、形成意義的一般性作用——更進一步，擴張為透過該作用使主體世界與鼎自身持續被形成的動態過程。始終明示何處為止是白川、自何處起是本體系。

### 補：五字的祭祀性布置【象徵性解讀】（於《常用字解》確認）

根據《常用字解》，本體系的中核五字皆為祭祀場域之字。

- **主**：象形。燈火火焰之形（《說文》「鐙中之火主也」）。在祭祀祖先的宗廟中手執火者為氏族長老之職責，持有神聖之火者即成為氏族與家的中心、主宰事物之人。因而為「主、主人」。
- **客**：會意。降臨於廟（宀）之中而至的神、自他處迎來的神（客神）＝賓客。其後轉為人間的客人・旅人之意。
- **觀**：透過鳥占察知神意之事。「看、審視」。
- **事**：會意。在史（捧持繫有祝詞之器的木枝以祭祀祖先）之上再繫上飄帶，於山河舉行國家性的祭祀。「祭祀」→「政事、事、侍奉」。
- **鼎**：祭祀之器本身。

主（於廟內執火的中心）與客（自外造訪廟宇的神），作為同一祭祀場域的內與外，自始便構成一對。本體系的整套用語，重構了一個祭祀的場景——主執火、客神造訪、透過觀察知神意、作為事而舉行祭祀、鼎作為其器而立。這並非論證，而是布置的解讀，作為【象徵性解讀】而提出。

另外關於「物」，白川《常用字解》並列①以耒（犁）翻土之形、②犧牲之牛毛色各異（《詩經》）、③旗幟標記（《說文》）三種說法而未定論（→docs/glossary.md 的字源註記）。

## 十、關於「不存在完全一致之既有概念」的表述

就目前所查，在主要的哲學・宗教・語言體系（西方哲學・佛教・儒教・道教・現象學，英・德・希・梵等）之中，未能找到與觀（Kan）＝「主體認識世界的作用本身，並透過該認識使主體世界被形成・變化」，以及客鼎觀＝「他者之觀」完全一致的既有概念。

**表述上的規律**：不斷言「任何語言中都沒有」。僅止於「就目前所查，在主要體系中未能找到」。針對全世界所有語言斷言「沒有」，是任何人都無法做到的。

## 十一、關於量子力學

與「透過觀測而確定顯現方式」此一構想的對應，可以作為思想上的類比來陳述。但不說「量子力學已加以證明」。
