# 與般若心經的對照 — 調和的主觀主義 v0.1.1

**定位**：般若心經並不**證明**本體系的正確性。般若心經是一種**檢驗裝置**——過去是作者在讀經。從此處起則相反，**是體系在讀經**。體系能否對經文全文檢驗出對應關係，構成了體系的強度測試。同時，經被置於思想的起源、對照對象、以及**自我批判裝置**（尤其是無智亦無得）的位置。對照素材為作者所作的般若心經白話譯文。

---

## 一、觀自在——經的第一語【詮釋】

經以「觀自在菩薩」——**自在運作「觀」者**——為開端。此處的「觀」是 avalokita（深遠地遍觀、洞見、觀察）的譯語，並非單純的視覺，而是以智慧深深映照的作用。本體系將此「觀」視為相同的義，並向上一層擴展為「一切認識的根源運動」（明示此為擴展）。

### 觀的可操作性——人類能否操控觀（Kan，中文讀音 guān）

**能操控。但這是程度的問題，且可經訓練。**其根據正是系三（自由意志）本身：指向（注意的指向）、接收（對所遞出之客鼎觀（Kyakuteikan）的審視）、遞出（主鼎觀（Shuteikan）的提供）——此三個動詞即為觀之操作的三種型態。完全的操作（連第一拍都加以選擇）並不可能，但作為參與形成的操作則是可能的。

佛教以此可操作性為前提，建立了修行體系——止觀正是字面意義上的**觀之訓練**。而**觀自在菩薩，可讀作觀之可操作性的極限意象**——完全自在地運作觀者的理想形態。人類的觀之操作是局部的，觀自在則是極限。兩者並非斷裂，而是程度上的連續。

## 二、五蘊＝鼎（Kanae，中文讀音 dǐng）的解剖圖【詮釋】

五蘊（色、受、想、行、識）在傳統上是人＝主體之構成要素的清單。因此五蘊可讀作鼎（Kanae）的解剖圖。※以下為假設性的對應，全文檢驗仍是課題。

| 蘊 | 傳統意義 | 於體系中的對應 |
|---|---|---|
| 色 | 物質 | 鼎的物質性相位（身體、硬體）。在觀之中作為物（Mono，中文讀音 wù）而顯現的一側 |
| 受 | 感覺 | 觀（Kan）的**接收**位相——刺激、客鼎觀（Kyakuteikan）的接受 |
| 想 | 認識、表象 | 觀（Kan）的**固定**位相——物化。截取流動、賦予名稱、將其立為「同一者」的作用 |
| 行 | 意志、形成作用 | 觀（Kan）的**形成**位相——相互形成之定理本身 |
| 識 | 意識 | 觀（Kan）的**識知**位相——基底的認知 |

**結論**：觀（Kan）內含受、想、行、識四個位相。然而鼎（Kanae）的形成並非僅由觀而成——形成有兩個系統：①由觀而來的自我形成（行的位相），②由事（Koto，中文讀音 shì）而來的形成（他鼎、環境、身體性過程）。觀承擔前者，並作為接收後者的窗口（介面）。

## 三、色即是空的導出【定理／詮釋】

從「物（Mono）＝觀（Kan）自變化之流中所截取的固定顯現」此一第三原理出發：

- 物既是觀所作的截取，故其自身並無固定的本性 → **色即是空**
- 流動唯有透過觀才能作為物而顯現 → **空即是色**

「色即是空、空即是色」此一經文的核心句，作為定義的必然結果而被再現。固定性並非世界一方的性質，而是觀的工作——此一點即是這項導出的關鍵。

## 四、照見五蘊皆空——對鼎之持續的回答【詮釋】

當觀自在菩薩將深邃的觀朝向五蘊本身（＝鼎的構成要素）時，並未尋得固定的核心（照見五蘊皆空）。

這構成了經典一方對「鼎（Kanae）是否持續存在」的回答：**鼎的持續並非實體的持續，而是形成的連續性。**如同火焰持續燃燒、河川持續流動，在沒有同一內核的情況下連續。無我的是自我（某一時點的形態＝作為物的顯現），連續的是鼎（形成的過程）。

## 五、無智亦無得——體系的最大難關（自我適用）【詮釋】

經將否定折返至觀本身：「智慧與應得之物皆不存在（無智亦無得）」。

以體系的語言來說：**不可將觀（Kan）當作物（Mono）來把捉。**此處須求精確——並非「一旦定義觀，觀便不再是觀」。**觀的定義是截取（物），而非觀本身。**地圖是截取，而大地持續流動。製作定義是被允許的。被誡止的，是將定義誤認為觀本身而加以把捉。

### 空亦復空、二諦——兩千年前的先行解

此一自我指涉的結構，與「一旦定義空，空便不再是空」完全同型，而佛教早已握有答案。

- **空亦復空**（龍樹、中觀）：空亦復是空。將空當作實體而把捉的瞬間，空便不再是空。因此，對空這一概念本身亦施以空。於本體系中可如此書寫：**觀的定義，亦是觀所作的截取。**
- **二諦**：於世俗諦可立言語與定義。於勝義諦則不將其固定。透過區分兩個真理的層次，「言說卻不把捉」得以成為可能。

**帶有版本的公開，即是二諦的現代性實作。**v0.1.1 是世俗諦的截取，確定版永遠不會到來。「正因無所得，故心無罣礙」——定義作為形成的中途過程而被提示，並不被固定。體系的最大難關，早在兩千年前便已有先行解。

## 六、尚待對照的課題

- 「無眼耳鼻舌身意、無色聲香味觸法」——六根、六境於體系內的對應（是否為受之位相的細分）
- 「無無明亦無無明盡、乃至無老死亦無老死盡」——十二因緣的否定，以及緣起、因果於體系內的定位
- 「無苦集滅道」——四諦之否定的解讀
- 真言（揭諦揭諦波羅揭諦…）——「能行者啊，往彼岸去吧」該如何於體系中承接（形成的方向性？對調和的呼喚？）
