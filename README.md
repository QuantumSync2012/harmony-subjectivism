## 本体系の性格

**本体系は「主観論」ではなく「形成論」である。**

中心命題は「人は世界を形成する」ではない。**「世界を形成することによって、鼎（Kanae）も形成される」**であり、さらに**「鼎（Kanae）同士は互いを形成し合う」**である。世界は完成したものではなく、形成され続ける。ゆえに調和とは完成状態ではなく、**形成の様式**である。

## 語義の独立宣言（最重要の前提）

**本書の語は、辞書的・哲学史的な既存の語義を参照しない。各語の意味は、本書の定義のみが与える。**

字源については、主として白川静『常用字解』で確認した記述を出発点とし、『字通』『字統』も参照する。ただし、白川説を絶対視せず、一学説として扱う。

## 表記規則（全文書共通）

- 鼎（Kanae）・観（Kan）・物（Mono）・事（Koto）は、定義・初出時に必ず漢字＋ローマ字を併記する
- 鼎の読み：単独では **kanae**、熟語内では **tei**（鼎談・鼎立と同じ音便）
- 正式語と略記：**主鼎観（Shuteikan）／以下、主観**・**客鼎観（Kyakuteikan）／以下、客観**と冒頭で宣言し、以後の「主観」「客観」はすべて略記として読む
- 観の提供・観の受領に固有の語は作らない。「観（Kan）を差し出す」「観（Kan）の提供」という句で表す（分かりづらい語を増やさない方針）
- 英語版：Kanae / Kan / Mono / Koto / Shuteikan / Kyakuteikan は翻訳せず、初出時に定義を添える。英語略記は Self-Kan / Other-Kan。体系名は **Harmonious Subjectivism** に統一
